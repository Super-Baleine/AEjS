#AEjS

## Description
AEjS is a implementation of the AES (Rijndael) algorithm in Javascript

## How to use ?
__Encrypt :__
```javascript
// Key length can be 128/192/256 bits (higher is better)
var keyLength = 256;

var cipherText = aes.actions.encrypt("My private content", "My p@ssW0rD", keyLength);
```
